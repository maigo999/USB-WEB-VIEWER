# USB WEB VIEWER
## 概要
Chrome61から実装されたWEBUSBAPIを使用して　ブラウザ上でUSBデバイスのVendorID、ProductID、SerialNumberを調べることができるもの。
## 制作理由
ソフトを入れずともIDを調べることができるのでWEBUSBAPIの開発を始めるハードルを下げれれば良いなと考え制作しました。
## 苦労点
日本でほとんど広まっていない技術だったので情報が少なかった。
## 制作日
2018年2月

1. HTML
2. CSS
3. javaScript
